## 工作规则

### 分析流程

1. **阅读源码**：使用 CodeGraph 辅助理解
2. **查阅文档**：参考官方文档和注释
3. **整理输出**：按规范撰写学习文档
4. **交叉验证**：确保理解准确

### 文档规范

- 每篇文档职责单一，只讲一个主题
- 代码示例必须可编译（或标注出处）
- 复杂流程使用 Mermaid 图表
- 术语首次出现时给出定义

### 质量检查

- [ ] 内容准确，无技术错误
- [ ] 结构清晰，易于理解
- [ ] 代码示例可运行
- [ ] 引用来源明确
