## 文档规范

### 格式要求

1. **标题层级**：最多 3 级（H1 → H2 → H3）
2. **代码块**：必须标注语言（```rust）
3. **表格**：用于结构化对比
4. **图表**：复杂流程使用 Mermaid 图

### 内容要求

1. **学术精简**：避免冗余，直击要点
2. **信息完备**：单篇文档自成体系
3. **职责单一**：一篇文档只讲一个主题
4. **可验证**：代码示例必须可编译

### 命名规范

```
{序号}-{模块名}.md
示例：04-executor.md、12-gpio.md
```
